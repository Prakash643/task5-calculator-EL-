Modular Calculator using Python (Functions & Modular Programming)

Project Description

This project is a menu-driven modular calculator developed using Python.
It demonstrates the use of functions, parameters, return values, default arguments, and modular programming concepts.

The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division, while handling runtime errors like division by zero.

Objectives

Understand function creation and reuse

Learn modular programming design

Practice user input handling

Implement error handling

Write clean and maintainable code

Tools & Technologies

Language: Python 3

Editor: VS Code / PyCharm / IDLE
